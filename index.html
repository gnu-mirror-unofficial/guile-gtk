Guile-gtk is glue code to make the GTK+ (the Gimp Toolkit) accesible from Guile so it provides a convenient interface for Scheme programmers to develop visual applications. It can optionally integrate with gnome via the Gnome-guile module.

Guile-gtk has been started by Marius Vollmer and is currently mantained by Ariel Rios and other intrepid Guile hackers around the world. Guile-gtk is free software licensed under GPL.

Current stable version is 0.19 and the latest snapshot can always be found in the Savannah guile-gtk project.

There is also the guile-gtk-1.2 series that offers stable support for Gtk+-1.2 only. The main line of guile-gtk evolves to cover Gtk+-2.0 and Gnome.
