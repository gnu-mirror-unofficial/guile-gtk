Gtk GL Module

The Gtk GL interface module can be used with 

     (use-modules (gtk-1.2 gtk-gl-area))
     (use-modules (gtk-1.2 gdk-gl))

The following rules apply to the various Gtk GL types.

Attributes
The attributes for gtk-gl-area-new and similar are taken as a list of symbols and integers. For example, 
          (gdk-gl-choose-visual '(doublebuffer buffer-size 123))
