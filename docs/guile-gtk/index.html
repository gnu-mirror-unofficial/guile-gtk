guile-gtk-1.2 documentation

Guile-Gtk User Manual: HTML

The guile-gtk-1.2 package is a continuation of the original guile-gtk code base. When Gtk+ 2.x appeared on the scene, trying to support both Gtk+ 1.2 and the new Gtk+ 2.0 in one version of guile-gtk was not fully satisfactory.

People wanting to have a stable but evolving platform for programming with Gtk+ 1.2 in Guile Scheme often had problems that were caused by the Gtk+ 2.0 support in guile-gtk. Likewise., supporting Gtk+ 2.0 could not be done the Right Way since one had to keep the legacy Gtk+ 1.2 in mind.

That's why the guile-gtk-1.2 project was created. The support for Gtk+ 2.0 and Gnome can be found in the guile-gobject project.

Documentation for guile-gtk-1.2 is available in the distribution itself, and also online.
