documentation

There are a number of sub-projects within the guile-gtk project. Each one has its own codebase and design goals. So, to get documentation, you need to first choose which branch of development you're interested in.

guile-gobject

The line of development supporting GNOME 2.x. Includes bindings for GObject, GLib, Pango, Atk, Gdk, and of course Gtk.

guile-gtk-1.2

The line of development supporting Gtk+ 1.2, including bindings for Gtk, Gdk, Glade, GdkPixbuf and GtkGL. This is a continuation of the original guile-gtk code base and is intended to be stable and independent of Gnome.

gnome-guile

The old line of development supporting GNOME 1.x, now (mostly?) dead.
