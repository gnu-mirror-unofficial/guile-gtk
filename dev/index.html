Savannah project page

guile-gtk is graciously hosted on the Free Software Foundation's savannah server. Our project page links to more information on our CVS repository and features other tools such as a bug tracker.

Our CVS repository can be checked out with 

cvs -z3 -d:pserver:anonymous@cvs.savannah.gnu.org:/sources/guile-gtk co guile-gtk-1.2
There's no password, you don't have to do a "login". Change into the guile-gtk-1.2 directory and run ./autogen.sh to apply autoconf and friends to get something ready to configure etc. (You need Autoconf, Automake and Libtool.)

You can also browse the project's CVS on the web.
