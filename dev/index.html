developer information

Savannah project page

guile-gtk is graciously hosted on the Free Software Foundation's savannah server. Our project page links to more information on our CVS repository and features other tools such as a bug tracker.

Our CVS repository can be checked out with cvs -d :pserver:anonymous@subversions.gnu.org:/cvsroot/guile-gtk/ co guile-gtk. Change into the guile-gtk directory and run ./autogen.sh to build the code.

You can also view the projects's CVS on the web.

guile-gobject Arch repository

guile-gobject has switched primary development from CVS to GNU Arch, which offers greater flexibility and more powerful features. If you want to hack on guile-gobject, you are strongly encouraged to take a look at Arch. CVS access to guile-gobject is also provided, but is more-or-less intended as a read-only snapshot of the Arch repository.

Archive name: guile-gnome-devel@gnu.org--2004
Archive location: http://people.debian.org/~rotty/arch/guile-gnome-devel@gnu.org/2004/

To check out the latest development version directly from the archive, run 

       tla register-archive http://people.debian.org/~rotty/arch/guile-gnome-devel@gnu.org/2004/
       tla get guile-gnome-devel@gnu.org--2004/guile-gobject--dev--0 guile-gobject
Then change into the guile-gobject directory and run ./autogen.sh to build the code.

To check out a released version, use a release branch, e.g.: 

       tla get guile-gnome-devel@gnu.org--2004/guile-gobject--release--0.5--patch-3 guile-gobject-0.5.3
for the 0.5.3 release.

The dev--0 branch always targets the latest (unstable) GNOME version; when a new stable GNOME is released, a corresponding main branch is created - currently, there is main--2.4 which targets the GNOME 2.4 series.
