Savannah project page

guile-gtk is graciously hosted on the Free Software Foundation's savannah server. Our project page links to more information on our CVS repository.

Our CVS repository can be checked out with cvs -d :pserver:anonymous@subversions.gnu.org:/cvsroot/guile-gtk/ co guile-gtk. Change into the guile-gtk and run ./autogen.sh to build the code.

You can also view the projects's CVS on the web.
